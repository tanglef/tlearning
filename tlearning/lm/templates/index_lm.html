What is a linear model?

The shortest way to describe the linear model is through a short example. Suppose you want to sell \(10\) video games and wish to know how much you'll gain. This price will depend on:

  • the base price for \(10\) games,
  • how many rare games there are amongst them (here all rare games sell at the same value).
If a rare game is a bonus of \(\$100\) and the set of \(10\) games is bought at \(\$120\), then for \(2\) rare games in your possession you've got yourself $$120 + 2\times 100 = \$320\enspace \text{(not bad!)}$$ Most of you are currently screaming at the screen (with right) thinking that it's way too over simplified. AND IT IS! That is actually one of the best assets of the linear model: it's a very simple vision of a complex situation.

So it's simple. But let's visualize our benefits if we decided to keep some common games and sell more rares gammes.

Of course, the more rare games you have in the \(10\), the best price you will get. So for a set of totally common games, you'll end up with \(\$120\) and for a set only made of rare games you gain \(\$ 1120\). See the very straight line? It's actually the line with the equation $$y=120+100x\enspace.$$ This equation is the perfect model for the prices here.

But until here we only asked the seller how much they would buy our games at the current time. They made an estimation but it may not be absolute. Let's play with that next.

With a little more math

Let's suppose you have some data \(X\in\mathbb{R}^{n\times p}\) meaning that you have \(n\) points with \(p\) features. Associated to each data point \(x_i\) is a reponse \(y_i\) contained in \(Y\in\mathbb{R}^n\). Then what you do with the linear model is looking for the vector \(\beta\in\mathbb{R}^p\) such that $$Y=X\beta\enspace.$$
Of course, most of the time the data isn't linear.
