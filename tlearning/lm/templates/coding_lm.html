There are several packages that can be used to make the linear regression in Python. However, because using a package can have a black box effect sometimes, let's try first to make it ourselves.

From scratch

To make a linear regression ourselves, we use:

  • the standard format in machine learning methods: using a class with a fit and a predict method (if you don't know how the class work in Python you can see this tutorial from the official documentation or this one by Jake VanderPlas),
  • the numpy package to make the matrix products and pandas for the data manipulation,
