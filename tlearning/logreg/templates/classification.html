Binary classification with more variables

The softmax function

As of now, we've only worked with \(1\) variable with a number of categories, which is quite limited. Let's make another step towards the generalization and consider \(p\) variables \(x_1,\dots x_p\) in a dataset of \(n\) rows. The goal is to predict the category assigned to a subject (for example \(1=\)dead, \(0=\)alive if the interest if the prediction of the death). So, our new model is: $$\mathbb{P}(Y=1\,|\, x)=\frac{1}{1+e^{-\beta' x}}\enspace ,$$ where \(\beta'x=\beta_0 + \beta_1 x_1 + \dots + \beta_p x_p\).

When using this model in practice, most of the time you don't need to add a columns of \(1\)'s in your dataset for the intercept \(\beta_0\). It may come as an argument or is simply estimated automatically. The exception being when using the statsmodels package in Python.

As we're used to now, the goal is to maximise the log-likelihood of our model to estimate \(\beta\).
